Core Research Areas

Arctic Methane Inverse Modeling

Geostatistical inversions • Bayesian inference • Uncertainty quantification

Quantifying methane emissions across Arctic-boreal North America using physics-based inverse modeling. Combining aircraft and tower observations with atmospheric transport models (e.g., STILT) to estimate regional fluxes and reduce uncertainty in emission budgets.

Impact

  • Reduce uncertainty in regional methane budgets
  • Diagnose seasonal and driver controls on emissions
  • Constrain wetland feedback under rapid warming
  • Translate sparse observations into robust estimates

Methods

  • Geostatistical inverse modeling (GIM)
  • Transport footprint calculations
  • Bayesian inference & optimization
  • Model evaluation against observations
  • Scalable HPC computation

Technical Stack

  • Python (NumPy, SciPy, PyTorch)
  • MATLAB for legacy workflows
  • HPC cluster computing
  • Reproducible analysis pipelines
  • Uncertainty quantification frameworks

Model Evaluation & Data Fusion

Bias diagnostics • Seasonal attribution • Multi-model comparison

Systematic evaluation of process-based methane models against observational constraints. Identifying regional biases, seasonal patterns, and interannual variability across North American ecoregions.

Approach

  • Multi-model ensemble comparison
  • Bias diagnostics across ecoregions
  • Seasonal and interannual analysis
  • Uncertainty propagation

Applications

  • Inventory validation
  • Model improvement guidance
  • Policy-relevant insights
  • Research prioritization

ML as a Research Accelerator

Surrogate models • Feature selection • Hybrid physics-ML

Using machine learning strategically to accelerate expensive components and improve representation where appropriate. Building hybrid physics-ML systems that maintain interpretability while gaining computational efficiency.

Applications

  • Transport model emulators
  • Emission pattern recognition
  • Feature selection for priors
  • Uncertainty estimation

Philosophy

  • ML as a tool, not a replacement
  • Maintain physics-based constraints
  • Validate against observations
  • Interpretable, reproducible workflows

Technical Methodology

Production-ready approaches that scale from research prototypes to validated systems.

Probabilistic frameworks for parameter estimation, uncertainty propagation, and posterior analysis. Quantifying confidence in model predictions.
