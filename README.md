# Hanyu (Mason) Liu - Personal Website

**Live Website:** [https://hanyumasonliu.github.io/](https://hanyumasonliu.github.io/)

## Features

- **Multi-page structure** with dedicated sections for Research, Experience, Tennis, and Contact
- **Responsive design** optimized for desktop, tablet, and mobile devices
- **Dark mode toggle** with system preference detection
- **SEO optimized** with meta tags, Open Graph, and structured data (JSON-LD)
- **Accessibility** compliant with semantic HTML, ARIA labels, and keyboard navigation
- **Fast performance** with minimal JavaScript and optimized CSS
- **GitHub Pages** compatible static site

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom design system with CSS variables
- **JavaScript** - Minimal vanilla JS for dark mode and navigation
- **Font Awesome** - Icon library (CDN)
- **GitHub Pages** - Hosting
